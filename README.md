# Advanced 2D Platformer - Unity

## Description
This is an improved 2D platformer game developed in Unity using C#. It builds on a previous project with enhanced gameplay systems, smarter enemies, and better overall design.

## Features
- Smooth and improved player movement  
- Smarter enemy AI with better movement behavior  
- Collectible items system  
- Parallax scrolling background for depth effect  
- Upgraded UI design and layout  
- Improved level design and gameplay flow  
- Additional gameplay systems and mechanics  

## Technologies Used
- Unity Engine  
- C#  
- Visual Studio  

## Purpose
This project was created to improve upon previous work and practice:
- More advanced Unity 2D development  
- Better enemy behavior  
- UI/UX design in games  
- Game feel and polishing mechanics
